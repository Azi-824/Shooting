ゲーム説明
・操作はマウスで行う。
・ゲームプレイ中は、画面に表示される的をクリックすることでスコアが加算される。
・ぬいぐるみは、スコアの加算、ハートと王冠は、時間の回復、爆弾は、時間とスコアがマイナスされる。
・制限時間内にハイスコアを目指そう！

開発環境 Windows10 / Visual Studio 2017

制作期間 4日間(約25時間30分)

使用したライブラリ DxLib

制作で特に頑張った部分
・Sceneクラスを基にした、各シーンを作成することで、main.cppの79行目の一行で現在のシーンごとの処理を行う、多態性を持たせることができた。
・的を、おもちゃ、爆弾、アイテムの三種類用意したが、それぞれクリックされた時の処理をEvent()メソッドでオーバライドすることで、種類ごとに別の処理を行わせることが簡単にできるようにした。
・タイトル画面の、「スタート」と「エンド」のボタンをクリックしたときの処理を、title.cppの69行目から、95行目で、設定しているが、ラムダ式を使うことで、ボタン毎の処理を簡単に設定できるようにした。
